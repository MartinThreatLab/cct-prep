Nmap — Network Mapper

Herramienta de descubrimiento de red y auditoría de seguridad

$ nmap -sS <target>
SYN Stealth Scan (half-open). El más usado. Requiere privilegios root.
Reconocimiento
$ nmap -sV <target>
Detecta versiones de servicios corriendo en los puertos abiertos.
Enumeración
$ nmap -O <target>
Detección de sistema operativo (OS fingerprinting via TCP/IP stack).
Reconocimiento
$ nmap -A <target>
Scan agresivo: OS detection + service versions + script scan + traceroute.
Enumeración
$ nmap -sU -p 53,161 <target>
UDP scan en puertos específicos (DNS=53, SNMP=161). Más lento que TCP.
Reconocimiento
$ nmap -p 1-65535 <target>
Escanear todos los puertos (1-65535). Completo pero lento.
Reconocimiento
$ nmap -sn 192.168.1.0/24
Ping sweep: descubrir hosts activos en una subred (sin port scan).
Reconocimiento
$ nmap -sF <target>
FIN scan: envía paquetes TCP FIN. Útil para evadir firewalls stateless.
Evasión
$ nmap --script vuln <target>
NSE Script: ejecuta scripts de detección de vulnerabilidades conocidas.
Vulnerabilidades
$ nmap -oX output.xml <target>
Guardar resultado en formato XML (compatible con Metasploit, etc.).
Utilidad

Diagnóstico de Red

Comandos esenciales de troubleshooting en Windows y Linux

> ipconfig /all
Windows. Muestra configuración completa: IPs, MAC, DNS, DHCP lease, gateway.
Windows
$ ifconfig / ip addr
Linux. Muestra configuración de interfaces de red. ip addr es el comando moderno.
Linux
> ping 8.8.8.8
Prueba conectividad usando ICMP Echo Request/Reply. Verifica si el host responde.
WindowsLinux
> tracert 8.8.8.8
Windows (traceroute en Linux). Traza el camino de paquetes, mostrando cada hop/router. Detecta dónde se pierden paquetes.
WindowsLinux
> netstat -ano
Windows. Muestra todas las conexiones TCP/UDP activas con sus PID. Ideal para detectar conexiones maliciosas.
Windows
$ netstat -tulnp
Linux. Muestra puertos TCP/UDP en escucha con el proceso asociado.
Linux
> arp -a
Muestra la tabla ARP: mapeo IP→MAC de dispositivos en la red local. Detectar ARP Poisoning.
WindowsLinux
> nslookup google.com
Consulta DNS: traduce hostname a IP. Diagnóstico de resolución de nombres.
WindowsLinux
$ dig @8.8.8.8 google.com
Linux. Consulta DNS avanzada. Puedes especificar el servidor DNS y tipo de registro (A, MX, NS, etc.).
Linux
> route print
Windows (ip route en Linux). Muestra la tabla de enrutamiento del sistema.
WindowsLinux

Forense Digital

Adquisición de evidencia, análisis de memoria y disco

$ dd if=/dev/sda of=/mnt/evidence.img bs=4096
Linux. Crea una imagen forense bit a bit del disco. Esencial para preservar evidencia sin modificar el original.
LinuxForense
$ md5sum evidence.img
Genera el hash MD5 de la imagen forense. Verificar integridad (Chain of Custody).
LinuxForense
$ sha256sum evidence.img
Genera hash SHA-256 (más seguro que MD5). Preferido en investigaciones actuales para verificar integridad.
LinuxForense
$ volatility -f memory.dmp imageinfo
Volatility Framework: identifica el perfil del sistema operativo del volcado de RAM.
ForenseMemoria
$ volatility -f mem.dmp --profile=Win10x64 pslist
Lista procesos activos en el volcado de RAM. Detectar procesos maliciosos o inusuales.
ForenseMemoria
$ volatility -f mem.dmp --profile=Win10x64 netscan
Muestra conexiones de red activas encontradas en la memoria RAM. Identificar C2 servers.
ForenseMemoria
$ strings evidence.img | grep -i password
Extrae cadenas de texto legibles de una imagen de disco y filtra por palabra clave.
LinuxForense
$ autopsy
Lanza Autopsy: GUI forense que analiza imágenes de disco, recupera archivos eliminados y examina metadatos.
LinuxForense
$ steghide extract -sf image.jpg
Extrae datos ocultos mediante esteganografía de una imagen JPEG (steganalysis).

Hashing & Criptografía

Verificación de integridad y funciones hash

$ echo -n "texto" | md5sum
Genera hash MD5 de un texto. MD5 = 128 bits. Vulnerable a colisiones. No usar para seguridad.
LinuxCrypto
$ echo -n "texto" | sha1sum
Genera hash SHA-1 = 160 bits. Deprecated: vulnerable a ataques de colisión. Reemplazado por SHA-256.
LinuxCrypto
$ echo -n "texto" | sha256sum
Genera hash SHA-256 = 256 bits. Estándar actual para integridad de archivos y contraseñas.
LinuxCrypto
> certutil -hashfile archivo.txt SHA256
Windows. Calcula el hash SHA256 de un archivo. También soporta MD5, SHA1, SHA512.
WindowsCrypto
$ openssl enc -aes-256-cbc -in file.txt -out file.enc
Cifra un archivo con AES-256-CBC usando OpenSSL. Te pedirá una passphrase.
LinuxCrypto
$ openssl genrsa -out priv.pem 2048
Genera un par de claves RSA de 2048 bits (clave privada). Criptografía asimétrica.
LinuxCrypto

Cracking de Contraseñas

Herramientas de auditoría de credenciales (uso ético)

$ john --wordlist=/usr/share/wordlists/rockyou.txt hashes.txt
John the Ripper: ataque de diccionario con rockyou.txt contra un archivo de hashes.
LinuxCracking
$ john --show hashes.txt
Muestra las contraseñas ya crackeadas por John the Ripper de un archivo de hashes.
LinuxCracking
$ hashcat -m 1000 -a 0 hashes.txt rockyou.txt
Hashcat: -m 1000 = NTLM, -a 0 = ataque diccionario. Usa GPU para velocidad máxima.
LinuxCracking
$ hashcat -m 0 -a 3 hash.txt ?a?a?a?a?a?a
Hashcat: -m 0 = MD5, -a 3 = brute force, mask ?a = any printable char. 6 caracteres.
LinuxCracking
$ hydra -l admin -P rockyou.txt ftp://<target>
Hydra: brute-force de credenciales FTP. Soporta SSH, HTTP, RDP, SMTP, etc.
LinuxCracking
$ ftp <target_ip>
Conectarse a un servidor FTP. Ingresar usuario y contraseña cuando se soliciten.
LinuxWindows

Linux — Seguridad y Administración

Comandos de hardening, permisos y logs

$ chmod 755 script.sh
Permisos: 7=rwx (owner), 5=r-x (group), 5=r-x (others). Propietario: lectura+escritura+ejecución. Resto: solo lectura y ejecución.
Linux
$ chmod 644 archivo.conf
Permisos: 6=rw- (owner), 4=r-- (group), 4=r-- (others). Típico para archivos de configuración.
Linux
$ chown user:group archivo
Cambia el propietario y grupo de un archivo o directorio.
Linux
$ cat /var/log/auth.log | grep "Failed"
Debian/Ubuntu. Revisa intentos de login fallidos en auth.log. Detectar brute-force SSH.
LinuxForense
$ cat /var/log/syslog
Log general del sistema en Debian/Ubuntu. Eventos de kernel, daemons y aplicaciones.
Linux
$ cat /var/log/messages
Log general en RedHat/CentOS. Equivalente a syslog en sistemas basados en Red Hat.
Linux
$ ps aux
Lista todos los procesos activos con usuario, PID, CPU y comando. Detectar procesos sospechosos.
Linux
$ ssh user@192.168.1.10
Conectarse a host remoto por SSH (puerto 22). Sesión de terminal cifrada.
Linux
$ ufw enable && ufw allow 22/tcp
Activa el firewall UFW en Ubuntu y permite SSH. Hardening básico de servidor Linux.
Linux

Wireshark — Filtros de Captura

Filtros BPF (capture) y Display Filters para análisis de tráfico

▶ ip.addr == 192.168.1.10
Display Filter: muestra solo tráfico desde o hacia la IP especificada.
Display Filter
▶ tcp.port == 80
Display Filter: filtra todo el tráfico TCP en el puerto 80 (HTTP).
Display Filter
▶ http.request.method == "POST"
Filtra peticiones HTTP POST. Útil para encontrar envío de formularios o credenciales.
Display Filter
▶ tcp.flags.syn == 1 && tcp.flags.ack == 0
Detecta SYN scans (Nmap -sS): paquetes SYN sin ACK = inicio de handshake sin completar.
Display Filter
▶ dns
Muestra solo tráfico DNS. Útil para detectar DNS tunneling o exfiltración.
Display Filter
▶ icmp
Filtra solo tráfico ICMP (ping). Detectar ping floods o ICMP tunneling.
Display Filter
$ tshark -i eth0 -w capture.pcap
TShark (Wireshark CLI): captura tráfico de la interfaz eth0 a un archivo .pcap.
Linux
